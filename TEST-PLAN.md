# Mine Platform — 功能测试方案

> 每次完成一批功能改动后，按此方案从上到下逐项验证。
> 测试账号见 AGENTS.md 第四章。
> 新增功能时，在对应章节补充测试用例，标注版本号。

---

## 使用说明

- **测试环境**：`http://121.43.127.52:3000`（或绑定的域名）
- **测试顺序**：建议按章节顺序执行，后面的测试依赖前面的登录态
- **标记方式**：`[ ]` 待测试 `[x]` 通过 `[!]` 发现问题（记录在章节末尾）

## T01 — 用户认证

### T01-1 登录
- [ ] 用 `admin@mine-cap.com / password` 登录，跳转成功，顶部显示用户名
- [ ] 用 `ajin-zhang@mine-cap.com / Ajin@2026` 登录，角色为 mine_enterprise

## T02 — 前台项目展示（index.html）

### T02-1 项目列表
- [ ] 首页能正常加载已发布项目列表（`/api/mine-projects/published`）
- [ ] 项目卡片显示：矿种、省市、开发阶段标签
- [ ] 项目卡片显示：储量、品位、面积核心数据标签（有数据时）
- [ ] 项目卡片显示：AI 评分
- [ ] 项目卡片**不显示**项目编号（code 字段）
- [ ] 标签全部为中文，无 `exploration`、`unknown` 等英文标签
- [ ] 没有数据的标签字段不显示空标签

### T02-2 筛选功能
- [ ] 按矿种筛选，结果正确
- [ ] 按省份筛选，结果正确
- [ ] 关键词搜索有效

## T03 — 项目详情页（project-detail.html）

### T03-1 未登录访问
- [ ] 能看到：基本信息（矿种/省市/阶段）、脱敏描述、AI 评分
- [ ] 看**不到**：完整项目名、联系方式、项目资料、AI 分析详情
- [ ] 看到"登录后查看更多"引导

### T03-2 investor 账号登录后访问
- [ ] 能看到：基本信息、AI 摘要评分
- [ ] **看不到**：项目资料列表、数据完整性提示、项目管理入口、AI 投资价值分析按钮
- [ ] 能提交投资意向表单

### T03-3 mine_enterprise 账号登录后访问
- [ ] 能看到：完整项目信息、项目资料列表、项目管理入口
- [ ] 投资意向提交入口**不显示**（内部用户不需要提意向）

### T03-4 admin 账号登录后访问
- [ ] 与 mine_enterprise 权限一致，能看到所有内容
